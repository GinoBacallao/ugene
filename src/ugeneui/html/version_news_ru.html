Изменения в UGENE версии 38

Редактор Выравниваний

  • Расширен функционал копирования и вставки.
  • Добавлена сортировка по длине и начальным гэпам.
  • Добавлена конвертация ДНК-выравнивания в РНК-алфавит.
  • При экспорте ДНК-выравнивания в амино-представление добавлен выбор таблицы трансляции.
  • Добавлены альтернативные опции обработки гэпов при экспорте ДНК-выравниваний в амино-представление.
  • Улучшено масштабирование текста на мониторах с большим разрешением.
  • Перереработан режим синхронизации выравнивания с деревьями.
  • Возвращён упрощённый режим копирования текущего выделения в выравнивании.

Редактор Последовательностей

  • Улучшена скорость работы в режиме авто-аннотирования.
  • Исправлены ошибки работы с аннотациями в многострочном режиме.
  • Улучшен парсинг координат при создании аннотаций для круговых последовательностей.

Остальное

  • Улучшена скорость и стабильность работы BLAST для маппинга ридов.
  • При работе со сборками по Сэнгеру добавлено автопозиционирование по старту выбранного рида.
  • Исправлены ошибки при выборе файла в Дизайнере Схем для Linux/GTK3.
  • Улучшена общая стабильность и скорость работы UGENE.
